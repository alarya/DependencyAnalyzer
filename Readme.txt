For adding files for analysis 
Server0:
-Add files to the folder Repository.
-For a new project make a new directory within Repository and place the source inside 
 that directory

Server1:
-Add files to the folder Repository1.
-For a new project make a new directory within Repository1 and place the source inside 
 that directory

The Dependency analyzer, by default, searches for source code recursively within repository


The Dependency analyzer, by default, analyzes only valid "*.cs" files.
